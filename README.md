# Profit and Loss Report (Excel | Power Query | Pivot Tables)

## Overview
This project focuses on building a comprehensive **Profit and Loss (P&L) report** for **AtliQ Hardware (FY 2022)** using Excel and its advanced data analysis features.  

The goal was to derive meaningful insights into revenue streams and expenses while automating repetitive data tasks.

---

## Key Features
- **Comprehensive P&L Report**  
  Developed a detailed report providing insights into revenue, costs, and overall financial performance.

- **Automated Data Cleaning**  
  Used **Power Query** to clean and transform raw data, reducing manual effort and saving time.

- **Advanced Calculations with DAX**  
  Implemented DAX measures to enhance flexibility, accuracy, and user experience in analysis.

- **Interactive Analysis with Pivot Tables**  
  Enabled dynamic filtering for better decision-making insights.

---

## Tools & Technologies
- Microsoft Excel

- Pivot Tables  
- DAX (Data Analysis Expressions)
